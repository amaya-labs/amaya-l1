
# Amaya L1 — Vision and Scope

## Infrastructure for a Moving World

Amaya L1 is a Philippine-focused research and development initiative exploring a permissioned Avalanche Layer 1 network for practical consumer applications, mobility infrastructure, marketplace settlement, and verifiable real-world systems.

Amaya L1 is not being developed as a meme-token network or as a general-purpose platform for unrestricted contract deployment.

The project follows a product-first approach: build useful applications, prove their requirements, and develop dedicated infrastructure only where it creates measurable value.

## Vision

Amaya L1 aims to provide a shared settlement and verification layer for approved applications operating across:

- movement and fitness
- real-item marketplaces
- transport and parking
- payment reconciliation
- verifiable documents and records
- future institutional integrations

## Core Infrastructure

### Amaya L1

A planned permissioned Avalanche L1 with:

- approved validators
- controlled smart-contract deployment
- configurable transaction fees
- sponsored-gas support for ordinary users
- dedicated network capacity
- transparent and verifiable transaction records
- EVM-compatible application development

### Amaya Testnet

The initial testing environment will be used to:

- operate validators
- test RPC connectivity
- connect supported wallets
- transfer the native TAMAYA test asset
- deploy test smart contracts
- connect Move+ and OneTap
- practise validator failure and recovery
- measure performance and infrastructure costs

### TAMAYA

TAMAYA is the planned native asset of Amaya Testnet.

TAMAYA:

- has no monetary value
- is not intended for sale
- is not an investment
- is used only for gas, transfers, and technical testing
- is separate from any future production AMAYA asset

## Primary Applications

### Move+

Move+ is a gamified fitness platform covering walking, running, cycling, digital gear, challenges, rewards, and other movement-based experiences.

Potential Amaya L1 uses include:

- reward-settlement proofs
- digital-gear ownership
- challenge and achievement verification
- marketplace settlement
- sponsored user transactions

Raw health information, GPS routes, personal profiles, and private user data will remain off-chain.

### Move+ Marketplace

Move+ Marketplace connects users with real products and digital utility.

Potential Amaya L1 uses include:

- merchant settlement records
- buyer and seller transaction proofs
- payment reconciliation
- reward distribution
- order-status verification

Product information, delivery addresses, and private customer records will remain in protected application databases.

### OneTap

OneTap is a transport and parking payment infrastructure prototype designed for:

- tap-card payments
- offline operation
- operator reconciliation
- transport settlement
- parking payments
- merchant and operator dashboards

OneTap is planned as the first infrastructure-focused application to demonstrate settlement batching on Amaya Testnet.

## Future Research Areas

Amaya L1 may later explore:

- payment settlement between approved organizations
- contractor and supplier payment proofs
- document authenticity and version verification
- institutional reconciliation systems
- public-sector pilot demonstrations using synthetic data

These are research areas only. No government, bank, remittance, or institutional partnership is currently implied.

## Development Principles

1. **Products first, network second.**
2. **Local testing before public testing.**
3. **Testnet before mainnet.**
4. **No valuable assets on experimental infrastructure.**
5. **Private and personal information remains off-chain.**
6. **Validators, RPC nodes, wallets, relayers, and treasury systems remain separated.**
7. **Security and recoverability are core requirements.**
8. **Partnerships and production readiness are never claimed without evidence.**
9. **Mainnet will launch only after technical, financial, security, and legal readiness.**

## Current Status

Amaya L1 is currently in:

- research
- planning
- public documentation
- architecture design
- local proof-of-concept preparation

Amaya L1 does not currently represent:

- a public mainnet
- a production payment network
- a public token offering
- a licensed financial service
- an official government system
- an approved remittance platform

## Long-Term Goal

The long-term goal is to determine whether Move+, OneTap, marketplace settlement, and approved payment infrastructure can benefit from operating on a dedicated Avalanche L1.

The decision to launch mainnet will depend on evidence collected from the testnet, including:

- real application requirements
- transaction performance
- infrastructure costs
- validator reliability
- security testing
- user demand
- partner demand
- regulatory and legal readiness
